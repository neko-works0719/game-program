# game-program

##  ゲーム概要
本作品は、プレイヤーが青い四角を操作し、足場をジャンプして進みながら緑のゴールを目指すパズルアクションゲームです。  
ステージごとに配置された足場の高さや位置を見極め、正しい順序で移動することがクリアの鍵となります。

##  ルール・終了条件
プレイヤーは青い四角を操作し、足場を利用してゴールに到達するとステージクリアとなります。  
ゴールに到達すると次のステージへ進み、最終ステージをクリアすると最初のステージに戻ります。

##  操作方法
- 左右移動：矢印キー（← →） または A / D キー  
- ジャンプ：上矢印キー（↑） または W キー
