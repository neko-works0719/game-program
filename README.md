# game-program

##  ゲーム概要
本作品は、プレイヤーが青い四角を操作し、足場をジャンプして進みながら緑のゴールを目指すパズルアクションゲームです。  
ステージごとに異なる足場の配置や高さを見極め、適切なタイミングで移動・ジャンプすることが求められます。

##  ルール・クリア条件
- プレイヤーは足場を利用してステージを進み、ゴールに到達するとクリアとなります
- ゴール到達後は次のステージへ進みます
- 最終ステージをクリアすると最初のステージに戻ります

##  操作方法
- 左右移動：矢印キー（← →） または A / D キー  
- ジャンプ：上矢印キー（↑） または W キー

##  実行方法
1. 本リポジトリをダウンロードします
2. index.htmlファイルをブラウザで開いてください

##  技術要素
本作品では、以下の要素を実装しています：

- 当たり判定によるプレイヤーと足場の接触処理
- 重力・加速度を考慮した物理的な移動処理
- クラスを用いたゲームオブジェクトの設計
